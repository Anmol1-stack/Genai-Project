# Older Baseline Attempts

These folders contain **earlier baseline experiments** that were conducted during the research phase of this project. They are **not part of the final implemented system** and are kept here for reference and reproducibility only.

| Folder | Description |
|--------|-------------|
| `Mistral_Output_Baseline_Phase2/` | Mistral-7B-Instruct baseline evaluation outputs (Phase 2) |
| `phase2_baseline_outputs_llama/` | LLaMA-based baseline evaluation outputs and checkpoints |
| `phase2_hybrid_outputs_20260425_083519/` | Hybrid (LLaMA + Mistral) pipeline outputs from early experiments |
| `Phase_2_Baseline_2_Hybrid_Colab.ipynb` | Hybrid baseline notebook (Colab version) |

For the final system, see [`../running_files/`](../running_files/).
For LLM baseline comparison (Zero-Shot vs Prompted), see [`../baseline_llm_comparison/`](../baseline_llm_comparison/).
